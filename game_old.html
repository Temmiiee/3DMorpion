Comment jouer

  • Cliquez sur un cube vide pour placer votre marqueur
  • Le joueur ROUGE (X) commence, puis on alterne avec le joueur BLEU (O)
  • Pour gagner, alignez 3 de vos marqueurs dans n'importe quelle direction (ligne, colonne, diagonale, ou à travers les faces)
  • Il y a 49 façons de gagner sur un cube 3x3x3
  • Utilisez les boutons de rotation pour voir le cube sous différents angles
